# Integration Runner (CloudEagle) - Sample Project

This is a sample Spring Boot project that demonstrates dynamic external system
configuration, a generic API caller, and a temporary user store. Use it as the
basis for the assignment deliverable.

## Run locally
- Build: mvn package
- Run: mvn spring-boot:run
- The app listens on port 8080.

## Sample DB entry
See src/main/resources/data.sql which inserts a sample Calendly config (replace token).
